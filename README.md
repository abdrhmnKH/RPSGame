# Rock Paper Scissors Game (C++)

A console-based Rock Paper Scissors game developed in C++. The player competes against the computer in multiple rounds, with random computer moves, score tracking, colored console feedback, and a final game summary.

## Features

- Play against the computer.
- Choose the number of rounds before the game starts.
- Random computer choices using `rand()`.
- Displays the result of every round.
- Keeps track of:
  - Player wins
  - Computer wins
  - Draws
- Displays the final winner after all rounds.
- Allows the player to play again without restarting the program.
- Changes the console color depending on the round result (Windows).

## Concepts Used

- C++
- Functions
- Enumerations (`enum`)
- Loops
- Conditional statements
- Random number generation
- Score tracking
- Console input/output
